# Medvědí stezka
Webová aplikace sloužící pro správu a vyhodnocování soutěží medvědí stezky více uživateli zároveň.

## Plánované funkce
- OCR detekce odpovědí z kartiček
- Správa soutěží
- Automatické vyhodnocování výsledků
- Správa uživatelů
- Možnost vyhodnocování více uživateli
- (tisk diplomů)

## Použité technologie
- Docker Compose
- Nginx
- ASP.NET Core
- Vue.js
- SurrealDB
